# Matplotlib & Seaborn — Advanced Chart Practice

A focused practice project applying Matplotlib and Seaborn to explore a real-world dataset through four core visualization categories: **distribution, comparison, relationship, and composition**. The goal was to reason through *which* plot answers a given question before writing any code, rather than defaulting to familiar chart types.

## Dataset

**Gym Members Exercise Dataset** ([Kaggle](https://www.kaggle.com/datasets/valakhorasani/gym-members-exercise-dataset))
973 records, 15 columns covering demographic data (Age, Gender), physical attributes (Weight, Height, BMI, Fat Percentage), workout metrics (Session Duration, Workout Type, Workout Frequency, Experience Level), and heart-rate/health indicators (Max/Avg/Resting BPM, Calories Burned, Water Intake).

## Tools

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## What's Covered

Each notebook question was approached by first identifying the type of insight needed, then selecting the plot that best represents it:

**Distribution**
- Spread of calories burned, age, and BMI across members
- Identifying skew and outliers

**Comparison**
- Average calories burned by workout type
- Session duration and workout frequency across gender and experience level

**Relationship**
- Weight vs. calories burned
- Avg BPM vs. fat percentage
- Max BPM vs. resting BPM
- Full correlation matrix across all numeric features

**Composition**
- Breakdown of workout type preference across the member base
- Gender-wise workout type distribution
